About Me

Hello, I'm ConsumerJunk, a wannabe application developer and an upper-end mediocre programmer. I've been programming since I was 10 or so when it started as a hobby with Scratch. But as my desire to want to do more complicated things grew, so did my knowledge of other programming languages. After a short stint with QBasic (I had a programming book about the language that was surprisingly popular at my school) I moved on to C#. I really liked C#, but found that I couldn't quite do what I wanted with it. The language itself was fine, but I never was fond of the way the code looked, and while there are plenty of YouTube videos on it, almost all of them don't know how to properly place their brackets. After my initial experience with these unsuitable languages (Except scratch, that'll always be my first) I found what really mattered in life. Not family, friends, money or even coffee. The real thing you can look forward to in life is the moment you discover Java.

ApolloStats

This project was designed to have two components. A Minecraft Bukkit plugin, and a web interface. The plugin sends data to a BitBucket repository as a CSV file, the file is then read via JavaScript and Ajax, and compiled into an interactive table.
